# film-site-data-archive
该仓库用于构建影视类网站的数据归档集合，对电影与电视剧站点的公开信息进行基础整理。项目强调信息结构统一，便于后续进行站点内容覆盖范围与更新节奏的分析。仓库不存储任何影视播放资源，仅作信息索引用途，所有资料均整理自公开网络。
